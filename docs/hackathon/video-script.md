# 视频脚本

## 开场

- 简介：OS_Agent 是一个面向服务器操作系统的智能代理
- 强调：CLI 是唯一交互入口，所有执行轨迹都会写入审计目录

## 第一段：CLI 基础能力

- 演示磁盘查询
- 演示文件检索
- 演示端口占用查询

旁白重点：

- 用户只输入自然语言
- 系统会自动生成结构化执行计划
- 每一步都能看到风险和结果解释

## 第二段：高风险控制

- 输入普通用户创建任务
- 展示高风险确认
- 再输入危险删除任务
- 展示阻断与解释

旁白重点：

- 不是简单执行命令
- 系统会先判断风险，再决定确认或拒绝

## 结尾

- 展示审计目录下的 `request.json`、`plan.json`、`events.jsonl`、`result.json`
- 总结：基础能力、风控能力、连续任务、可解释执行、可审计回放
